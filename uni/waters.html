📊 IFD / AEP Lookup Q A1, A4
Steps
  1. Sum total rainfall over the storm duration
  2. Compare total depth to IFD table for that duration
  3. Read off which AEP column it falls between
  4. For sub-bursts: sum rainfall for 5, 10, 15 min windows and compare to IFD
💡 The AEP of the storm = the most severe sub-burst AEP found
⚠ Use total depth (mm), not intensity — unless table says intensity (mm/hr)
💧 Losses & Runoff Volume Q A1, A2
Loss Model
Effective rainfall = Total − Initial loss − Continuing loss × duration
  • Initial loss applied once at start
  • Continuing loss = mm/hr × time increment (hr)
  • If effective rainfall < 0 in a period → set to 0
Runoff Volume
V = Σ(effective rainfall mm) × Area (m²) / 1000
  • Convert area: 1 ha = 10,000 m², 1 km² = 10⁶ m²
💡 Larger catchment → smaller areal average rainfall (spatial variability)
🌧 Temporal Patterns → Hyetograph Q A2d, A4d
Steps
  1. Get total storm depth from IFD table for the AEP & duration
  2. Multiply each time increment % by total depth → rainfall per period
  3. Apply initial loss first (subtract from early periods, stopping at 0)
  4. Then subtract continuing loss (mm/hr × period duration) each period
  5. Any negative result → 0 (no negative effective rainfall)
Depth per period = (% / 100) × Total IFD depth
💡 For "rare storms" temporal pattern → use the rare storms column from Table 2
⏱ Critical Storm Duration Q A2a
Definition

The storm duration that produces the highest peak flow at the catchment outlet.

From Box Plots (Figure 1)
  1. Look at ensemble average (★ star) for each duration
  2. Pick the duration with the highest ensemble average flow
  3. That is the critical duration
Which temporal pattern to give colleagues?

Give the pattern that produced the highest peak flow within the critical duration ensemble — i.e. the worst-case / most conservative single pattern.

🌡 Climate Change on Peak Flows Q A2e, A4c
Rainfall Scaling Rule (ARR)
ΔRainfall ≈ +5% to +10% per 1°C warming

Use ~5%/°C (conservative) or ~7%/°C (median) unless told otherwise.

Linear Catchment Assumption
Future Q = Current Q × (1 + ΔRainfall%)

Example: 3°C rise, 7%/°C:

ΔR = 3 × 7% = 21% → Future Q = 270 × 1.21 = 327 m³/s
⚠ IFD tables will need updating; existing infrastructure may be undersized
📈 Unit Hydrograph Convolution Q A5
Steps to get design hydrograph
  1. Scale UH by rainfall depth for each period
  2. Lag each scaled UH by the period number × UH duration
  3. Sum all lagged scaled UHs at each timestep
Q(t) = Σ [Pᵢ × UH(t − (i−1)×Δt)]
💡 Linearity: if rainfall doubles, peak flow doubles exactly
S-Curve Method (changing UH duration)
  1. Build S-curve: sum UH + lagged copies (lag = UH duration each time) until plateau
  2. Lag S-curve by the target new duration
  3. Subtract: S − S_lagged
  4. Scale by: (original duration) / (new duration)
New UH = (S − S_lag) × (D_old / D_new)
⚠ S-curve lag = original UH duration. New UH lag = target duration. Don't mix them up!
🗺 Thiessen Weights Q A6, A7
Concept

Each gauge is assigned a weight = fraction of catchment area closest to it (Voronoi polygon).

Weight = Area of polygon / Total catchment area
Σ weights = 1.0 always
Square catchment: centre + corner gauge

Draw perpendicular bisector between centre and corner. Area closer to centre vs. corner gives ~75% / 25% split (depends on geometry).

Catchment Average Rainfall Excess (A7)
  1. Compute weighted rainfall per period: R_avg = Σ(wᵢ × Rᵢ)
  2. Apply initial loss to weighted total
  3. Subtract continuing loss per period
  4. Sum remaining = rainfall excess
📡 RFFE / Regional Flood Frequency Q A3
Reading RFFE output
TermMeaning
Expected quantileBest estimate of peak flow for that AEP
5% CLLower bound (flow unlikely to be below this)
95% CLUpper bound (flow unlikely to exceed this)
Reporting uncertainty
Design Q = Expected ± [5%CL, 95%CL]

e.g. 2% AEP: Q = 373 m³/s (range: 163 – 857 m³/s)

💡 Use RFFE when no stream gauge at site. Uncertainty is large — always report confidence limits!
🔵 Hydraulic Head & Gradient Q B2, B3, B4
Hydraulic Head
h = z + p/(ρg)
  • z = elevation of piezometer intake (m ASL)
  • p/(ρg) = pressure head = water height above intake
  • h = water level elevation (m ASL)
h = Surface elevation − Depth to water
Hydraulic Gradient
i = Δh / ΔL (dimensionless)

Direction = from high head → low head

💡 Upward gradient (deep h > shallow h) = artesian / upward flow from aquifer to surface
🌊 Darcy's Law & Aquifer Flow Q B3, B5
Darcy's Law
q = −K · i (specific discharge, m/day)
Q = q · A = K · i · A (volumetric flow, m³/day)
  • K = hydraulic conductivity (m/day)
  • A = cross-sectional area = width × aquifer thickness
  • T = K × b (transmissivity, m²/day)
Flow through a section
Q = T · i · W

W = width of section (m), T = transmissivity (m²/day)

⚠ Equipotentials closer together → higher gradient → higher velocity (converging flow)

⚡ Bernoulli / Energy Equation Q B4
Total energy per unit mass
E = v²/2 + gz + p/ρ (J/kg)
  • v²/2 = kinetic (velocity) term
  • gz = potential (gravity) term; g = 9.80 m/s²
  • p/ρ = pressure term
💡 Groundwater velocity is tiny (~10⁻³ m/s) so v²/2 ≈ negligible (<0.001% of total)
